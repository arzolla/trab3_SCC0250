## Instruções:

Execute o arquivo ```/code/trab2.py``` para rodar o programa

**Controles:**

<kbd>Mouse</kbd> Para mover o ângulo da câmera

<kbd>P</kbd><kbd>U</kbd> Para aumentar/diminuir iluminação ambiente

<kbd>L</kbd> Para ativar iluminação colorida
